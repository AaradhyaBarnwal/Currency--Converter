<h1>Currency-Converter</h1>
A webapp where we can exchange currency rate.

<h1>How to use:</h1>
<ul>
  <LI>Enter your amount</LI>
  <li>Chose your currency from the options</li>
  <li>You can see the exchanged currency(All the currencies are arranged in alphabetical order.)</li>
</ul>
<h1>Features:</h1>
<ul>
  <li>It changes the flag when the currency is changed.</li>
  <li>It has a lot of options in the currency.</li>
</ul>
<h2>I made it using js , html and css.And I made it in the vs code editor.I used two api to make it.</h2>
